Company

Get All Companies

Get an array of all companies with a simple GET REQUEST.

https://byob-test.herokuapp.com/api/v1/companies

Get A Specific Company

To get a specific company, make a GET REQUEST with the specific company id.

https://byob-test.herokuapp.com/api/v1/companies/:id

Add A Company

To add a specific company, make a POST REQUEST with the body: { company_name: 'name', url: 'url', company_size: 'company size', job_openings: 00 }

https://byob-test.herokuapp.com/api/v1/companies

Update A Specific Company

To update a specific company, add the specific company id and make a PATCH REQUEST with the body: { company_name: 'name', url: 'url', company_size: 'company size', job_openings: 00 }

https://byob-test.herokuapp.com/api/v1/companies/:id

Delete A Specific Company

To delete a specific company, add the specific company id and make a DELETE REQUEST.

https://byob-test.herokuapp.com/api/v1/companies/:id

Jobs

Get All Jobs

Get an array of all jobs from every company with a simple GET REQUEST.

https://byob-test.herokuapp.com/api/v1/jobs

Add A Job

Add a job to a specific company with a POST REQUEST with the body: {title: 'title', company_id: 00, location: 'location name'}.

https://byob-test.herokuapp.com/api/v1/jobs

Get All Jobs At Specific Company

To get all jobs from a specific company, make a GET REQUEST with the company id as an address parameter.

https://byob-test.herokuapp.com/api/v1/jobs/:company_id/positions

Find Jobs Based On Location

To find jobs by location, make a GET REQUEST with an address query parameter.

https://byob-test.herokuapp.com/api/v1/jobs/find_by_location?location=denver

Update A Specific Job

https://byob-test.herokuapp.com/api/v1/jobs/find_by_location?location=denver
